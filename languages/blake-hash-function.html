BLAKE is a hash function created in 2012.

#1496on PLDB 10Years Old 685Users

BLAKE is a cryptographic hash function based on Dan Bernstein's ChaCha stream cipher, but a permuted copy of the input block, XORed with round constants, is added before each ChaCha round. Like SHA-2, there are two variants differing in the word size. ChaCha operates on a 44 array of words. Read more on Wikipedia...
