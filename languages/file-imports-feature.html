File Imports

File Imports

File Imports are a feature.

Many (most?) languages allow you to split code into different files and import the code from one file into a different file.

  • HTML of this page generated by LanguagePage.ts
  • Improve our File Imports file

Languages without File Imports include HTML, JSON

Languages with File Imports include Java, JavaScript, C, Python, C++, PHP, Perl, MATLAB, C#, Ruby, R, CSS, Go, Swift, Rust, Kotlin, Haskell, TypeScript, Clojure, Bash, Objective-C, Elixir, Node.js, Erlang, Julia, Dart, Crystal, F#, Racket, Ballerina, X10, Xtext, Slope, Jule, Flow9, Dale, Scroll
