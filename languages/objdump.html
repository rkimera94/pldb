ObjDump

ObjDump is an object object created in 1991.

#1334on PLDB 31Years Old 215Users

objdump is a program for displaying various information about object files on Unix-like systems. For instance, it can be used as a disassembler to view an executable in assembly form. It is part of the GNU Binutils for fine-grained control over executables and other binary data. Read more on Wikipedia...
