PDF, aka Portable Document Format, is a binary data format created in 1993.

The Portable Document Format (PDF) is a file format used to present documents in a manner independent of application software, hardware, and operating systems. Each PDF file encapsulates a complete description of a fixed-layout flat document, including the text, fonts, graphics, and other information needed to display it.. Read more on Wikipedia...
